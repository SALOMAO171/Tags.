Explorando Tags HTML

<h> - Títulos

A tag <h> é usada para definir cabeçalhos em HTML. Existem várias variações, como <h1>, <h2>, <h3>, etc., cada uma representando diferentes níveis de importância.

<p> - Parágrafos

A tag <p> é usada para definir parágrafos de texto em HTML. É usada para agrupar um bloco de texto.

<img> - Imagens

A tag <img> é usada para incorporar imagens em documentos HTML. Ela não possui uma tag de fechamento e requer atributos como src (para o caminho da imagem) e alt (texto alternativo para acessibilidade).
