Explorando Tags HTML e Propriedades CSS

<h> - Títulos

A tag <h> é usada para definir cabeçalhos em HTML. Ela possui várias variações, como <h1>, <h2>, <h3>, etc., cada uma representando diferentes níveis de importância.

O tamanho da fonte dos títulos pode ser ajustado usando a propriedade CSS font-size. Por exemplo, font-size: 24px; define um tamanho de fonte maior para <h2>.

<p> - Parágrafos

A tag <p> é usada para definir parágrafos de texto em HTML. Ela é essencial para estruturar e organizar o conteúdo textual em um documento web.

A propriedade CSS color define a cor do texto dentro de parágrafos e outros elementos. Por exemplo, color: #333; define um tom de cinza para o texto.

A propriedade CSS text-align pode ser usada para alinhar o texto dentro de elementos HTML. Por exemplo, text-align: justify; justifica o texto, enquanto text-align: center; centraliza o texto horizontalmente.

A propriedade CSS font-size define o tamanho da fonte dos parágrafos. Por exemplo, font-size: 16px; define um tamanho de fonte padrão para os parágrafos.

<img> - Imagens

A tag <img> é usada para incorporar imagens em documentos HTML. Ela não possui uma tag de fechamento e requer atributos como src (para o caminho da imagem) e alt (texto alternativo para acessibilidade).

Imagens podem ser centralizadas horizontalmente dentro de seu contêiner usando a propriedade CSS margin com margin: 0 auto;. Isso ajuda a garantir um layout visualmente equilibrado e esteticamente agradável.
