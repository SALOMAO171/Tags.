Explicação de Tags HTML e Propriedades CSS

Tag <h>

A tag <h> é usada para definir cabeçalhos em um documento HTML. Existem seis níveis de cabeçalhos, de <h1> a <h6>, sendo <h1> o mais importante e <h6> o menos importante. Por exemplo:

<h1>Cabeçalho 1</h1>
<h2>Cabeçalho 2</h2>
<h3>Cabeçalho 3</h3>

Tag <p>

A tag <p> é usada para definir parágrafos. É uma das tags mais básicas e essenciais no HTML para a estruturação de texto. Por exemplo:

<p>Este é um parágrafo de exemplo.</p>

Tag <img>

A tag <img> é usada para incorporar imagens em um documento HTML. Esta tag é auto-fechável e requer o atributo src para especificar o caminho da imagem. Por exemplo:

<img src="caminho/para/imagem.jpg" alt="Descrição da imagem">

CSS: Cor, Centralização e Tamanho

As propriedades de CSS permitem estilizar elementos HTML de diversas maneiras. Vamos ver algumas propriedades básicas:

Cor

A propriedade color define a cor do texto de um elemento. Por exemplo, no estilo acima, o texto dentro de <h1> é colorido usando color: #2c3e50;.

Centralização

A centralização de texto pode ser feita usando a propriedade text-align. Por exemplo, o texto dos cabeçalhos é centralizado com text-align: center;.

Tamanho

O tamanho do texto pode ser ajustado com a propriedade font-size. No exemplo, os parágrafos têm um tamanho de fonte definido por font-size: 16px;.
